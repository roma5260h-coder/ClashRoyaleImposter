## 🚀 НАЧНИ ОТСЮДА (3 минуты)

### 1️⃣ Прочитай README
```bash
cat README.md
```

### 2️⃣ Получи токен у @BotFather в Telegram
- Открой @BotFather
- Выполни `/newbot`
- Скопируй токен

### 3️⃣ Установи токен
```bash
cp .env.example .env
# Отредактируй .env и вставь токен
```

### 4️⃣ Установи зависимости
```bash
pip install -r requirements.txt
```

### 5️⃣ Запусти бота
```bash
./start.sh
# или
python3 main.py
```

### 6️⃣ Тестируй в Telegram
- Найди своего бота
- Отправь `/start`
- Нажми кнопку "Начать игру"
